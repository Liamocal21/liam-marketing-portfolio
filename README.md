# Liam O'Callaghan - Marketing Portfolio Website

A bold, creative, and interactive marketing portfolio website for Liam O'Callaghan, a recent marketing graduate from IMM Graduate School.

## Features

- **Modern Design**: Clean, visually appealing layout with vibrant colors
- **Responsive**: Works on all devices from mobile to desktop
- **Interactive Elements**: 
  - Animated statistics counters
  - Interactive background elements
  - Hover effects throughout
  - Smooth scrolling navigation
- **Sections**:
  - Hero/Introduction
  - About section with education highlights
  - Skills showcase with marketing tools
  - Academic projects portfolio
  - Contact form and information

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript (no dependencies)
- Font Awesome Icons
- Google Fonts
